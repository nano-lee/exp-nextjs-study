---
title: Data Fetching
description: Next.js allows you to fetch data in multiple ways, with pre-rendering, server-side rendering or static-site generation, and incremental static regeneration. Learn how to manage your application data in Next.js.
---

Next.js의 Data Fetching을 사용하면 애플리케이션의 사용 사례에 따라 다양한 방식으로 콘텐츠를 렌더링할 수 있습니다. 여기에는 **Server-side Rendering** 또는 **Static Generation**을 사용한 사전 렌더링, **Incremental Static Regeneration**을 사용한 런타임 콘텐츠 업데이트 또는 생성 등이 포함됩니다.
