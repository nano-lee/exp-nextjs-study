써드파티 라이브러리

@next/third-parties 라이브러리는 Next.js 앱에서 유명한 써드파티 라이브러리를 사용할 때 성능과 개발자 경험을 향상시킬 수 있는 컴포넌트와 유틸리티를 제공합니다.

토막 상식: @next/third-parties는 아직 개발중인 실험적인 라이브러리입니다. 현재 더 많은 써드 파티 라이브러리를 추가하고 있습니다.

@next/third-parties에서 제공하는 모든 써드파티 통합 기능은 성능과 사용자 편의성을 위해 최적화되었습니다.

시작하기​

먼저 @next/third-parties 라이브러리를 설치하세요.

npm install @next/third-parties

구글 써드파티​

모든 구글 써드파티 라이브러리는 @next/third-parties/google에서 가져올 수 있습니다.

구글 태그 매니저 (GTM)​

GoogleTagManager 컴포넌트는 페이지에 구글 태그 매니저 컨테이너를 초기화하는데 사용할 수 있습니다. 기본적으로 페이지가 하이드레이션된 후에 초기화를 위한 인라인 스크립트를 가져옵니다.

모든 경로에서 구글 태그 매니저를 로드하려면 이 컴포넌트를 커스텀 _app 파일에 직접 포함시키세요.

import { GoogleTagManager } from "@next/third-parties/google";

export default function MyApp({ Component, pageProps }) {
    return (
        <>
            <Component {...pageProps} />
            <GoogleTagManager gtmId="GTM-XYZ" />
        </>
    );
}

단일 경로에서 구글 태그 매니저를 로드하려면 로드가 필요한 각 페이지에 이 컴포넌트를 포함시키세요.

import { GoogleTagManager } from "@next/third-parties/google";

export default function Page() {
    return <GoogleTagManager gtmId="GTM-XYZ" />;
}

Nano's Comment

EXP Site _document.jsx에서 포함시키는 아래 스크립트에 대한 대안으로 보입니다.

<noscript>
    <iframe
        title="Google Tag Manager"
        src="https://www.googletagmanager.com/ns.html?id=GTM-XYZ"
        height="0"
        width="0"
        style={{ display: "none", visibility: "hidden" }}
        sandbox="allow-scripts"
    />
</noscript>

이벤트 보내기​

sendGTMEvent 함수는 dataLayer 객체를 사용하여 페이지에서 이벤트를 보내 사용자 상호작용을 추적하는 데 사용할 수 있습니다. 이 함수가 작동하려면 <GoogleTagManager /> 컴포넌트가 부모 레이아웃, 페이지, 컴포넌트 또는 동일한 파일에 직접 포함되어야 합니다.

옵션​

구글 태그 매니저에 전달 가능한 옵션 목록입니다. 전체 옵션 목록은 구글 태그 매니저 문서를 참고하세요.

NameTypeDescription
gtmIdRequiredGTM 컨테이너 아이디
dataLayerOptional컨테이너 초기화에 사용할 데이터 레이어 배열. 기본값: 빈 배열
dataLayerNameOptional데이터 레이어의 이름. 기본값: dataLayer
authOptional환경 구성을 위한 인증 파라미터 (gtm_auth)
previewOptional환경 구성을 위한 미리보기 파라미러 (gtm_preview)

구글 지도​

GoogleMapsEmbed 컴포넌트는 구글 지도를 페이지에 추가하는데 사용할 수 있습니다. 기본적으로 loading 속성을 통해 레이지 로딩을 사용합니다.

import { GoogleMapsEmbed } from "@next/third-parties/google";

export default function Page() {
    return (
        <GoogleMapsEmbed
            apiKey="XYZ"
            height={200}
            width="100%"
            mode="place"
            q="Brooklyn+Bridge,New+York,NY"
        />
    );
}

옵션​

구글 지도에 전달 가능한 옵션 목록입니다. 전체 옵션 목록은 구글 지도 문서를 참고하세요.

NameTypeDescription
apiKeyRequiredAPI 키
modeRequired지도 모드
heightOptional지도의 높이. 기본값: auto
widthOptional지도의 너비. 기본값: auto
styleOptionaliframe에 전달할 스타일
allowfullscreenOptional지도의 특정 부분이 전체화면으로 표시되도록 허용하는 속성
loadingOptional레이지 로딩이 기본값입니다. 지도가 항상 바로 노출된다면 값을 변경하는걸 고려하세요.
qOptional지도 마커 위치 지도 모드에 따라 필수값이 될 수 있습니다
centerOptional지도 뷰 중심 정의
zoomOptional지도 확대/축소 초기값 설정
maptypeOptional지도 타입 설정
languageOptional지도에 표시할 UI 요소 언어 정의
regionOptional정치적 민감도에 따라 적절한 국경과 라벨 정의

유튜브​

YouTubeEmbed 컴포넌트는 유튜브 영상을 페이지에 추가하는데 사용할 수 있습니다. 내부적으로 [lite-youtube-embed]를 사용하여 더 빠르게 로드합니다.

import { YouTubeEmbed } from "@next/third-parties/google";

export default function Page() {
    return (
        <YouTubeEmbed videoid="ogfYd705cRs" height={400} params="controls=0" />
    );
}

옵션​

NameTypeDescription
videoidRequired유튜브 영상 아이디
widthOptional영상 컨테이너 너비. 기본값: auto
heightOptional영상 컨테이너 높이. 기본값: auto
playlabelOptional접근성을 위해 재생 버튼에 숨겨진 레이블 표시 여부
paramsOptional이곳에 정의된 영상 플레이어 파라미터.
파라미터는 쿼리 파라미터 문자열로 전달.
예: params="controls=0&start=10&end=30"
styleOptional영상 컨테이너에 적용할 스타일
